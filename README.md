## Introduction

SuperRent merupakan website berbasis php framework Laravel versi 10, dikembangkan menggunakan XAMPP. Dibuat sebagai pemenuhan ketentuan penilaian UAS matakuliah Pemrograman Berbasis Website. SuperRent untuk saat ini masih belum dihosting, sehingga apabila ingin menjalankannya silahkan clone repo dan run di local host.

Website ini menggunakan Bootstrap 5, jQuery, dan Leaflet.js
